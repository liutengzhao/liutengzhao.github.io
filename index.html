如何打开电脑提示有病毒禁止打开的文件
发表于2024-05-17|更新于2024-05-14|项目部署|mysql
有时我们偏偏需要打开电脑提示有病毒的文件，但是即使把windows自带的防火墙、病毒监控、第三方杀毒工具等全关了，有些文件还是嗲不开，会提示有病毒禁止打开。这时候可以通过以下设置把文件排除在监控之外 进入 Windows 设置，点击【更新和安全】； 在更新和安全窗口中，先点击窗口左侧的【Windows 安全中心】，在右侧窗口中点击：打开Windows 安全中心； 在打开的 Windows 安全中心 - 安全性概览 窗口中，点击：病毒和威胁防护； 下拉窗口，找到【病毒和威胁防护设置】，点击：管理设置； 下拉窗口，找到【排除项】，点击：添加或删除排除项； 在 Windows 安全中心 - 排除项，我们点击：添加排除项，在下拉菜单中点击：文件夹，选择要排除的文件即可。
